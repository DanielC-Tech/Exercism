# My Python Journey on Exercism 🐍

This repository contains my solutions to the Python track on Exercism. Each folder represents a different challenge focusing on core programming concepts, data types, and idiomatic Python (Pythonic code).

Personal Notes
I am using these exercises to move from "knowing the syntax" to "thinking in Python." 

I focus on:
Writing clean, readable code according to PEP 8.
Ensuring all functions are properly documented with Docstrings.
Solving edge cases identified by the Exercism test suites.
